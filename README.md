#Py-To-res :fa-desktop:
## 依赖
```python
pip install requests beautifulsoup4 pillow
```
## 1. 网页文字提取功能

#### 从任意网页URL中提取纯文字内容，使用BeautifulSoup解析HTML，移除脚本、样式等非内容标签，显示干净、格式化的纯文本内容，快速获取文章内容、博客文字、新闻正文等

##  2. 图片识别与显示功能
#### 自动识别网页中所有图片并显示预览列表，查找HTML中所有`<img>`标签，提取图片URL和描述信息，支持相对URL转换为绝对URL，显示图片alt描述信息，自动统计图片数量


## 3. 图片查看器功能
#### 在新窗口中直接打开和查看选中的图片
#### 使用`PIL库`处理图片，支持多种图片格式
#### 自动调整大图尺寸，适应显示窗口
#### 保持图片原始宽高比显示图片原始尺寸和描述信息

## 4.组件：
#### URL输入框：输入目标网页地址
#### 标签页布局：文字内容和图片列表分页显示
#### 状态栏：实时显示操作状态

### 使用流程
#### 输入URL → 在文本框中输入目标网页地址
#### 获取内容 → 点击"获取网页内容"按钮开始处理
#### 查看文字 → 在"网页文字"标签页查看提取的文字内容
#### 浏览图片 → 在"图片列表"标签页查看所有图片缩略信息
#### 查看大图 → 双击图片项或点击"查看选中图片"打开图片查看器
